# Order.Land

Order.Land는 식당 주문 관리 시스템입니다. 웹 기반의 관리자 페이지와 모바일 앱을 통해 효율적인 주문 관리를 제공합니다.

## 주요 기능

- 실시간 주문 현황 모니터링
- QR 코드를 통한 테이블별 주문 시스템
- 메뉴 관리 및 카테고리 설정
- 매출 통계 및 분석
- 예약 관리 시스템

## 기술 스택

- **Frontend**: React + Vite
- **Mobile**: React Native + Expo
- **Backend**: Firebase (Firestore, Authentication)
- **Deployment**: Vercel

## 개발 환경 설정

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```
